About This Project

APO is a web interface for APT, a powerful software for Petri net analysis and synthesis. The first version is developed at the Carl von Ossietzky University in Oldenburg, Germany by Adrian Jagusch and was part of his bachelor thesis. See first version (2019).

This version contains new features developped by Emma Rizzi at the LAAS-CNRS, France as part of a 2 months internship (2020) during her 4th year at INSA Toulouse, France.

Once loaded, the editor is cached in the browser storage and can be used offline. Only for the analysis modules angular-apt (which was deleveloped as part of the same thesis) is used as a server side framework. Therefore an internet connection is needed.

To navigate in big nets using the browsers built in zoom functions (usually Cmd+'+', Cmd+'-' and Cmd+0) may help.
